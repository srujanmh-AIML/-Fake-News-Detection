# Fake News Detection System

This project detects whether a news article is Fake or True using:

- TF-IDF
- Logistic Regression
- Passive Aggressive Classifier
- Linear SVC
- Gradio Web Interface

## How to Run
In Ananconda Command Prompt 

conda create -n fakenews python=3.10

conda activate fakenews

cd C:\Users\SRUJAN M H\Downloads\Fake-News-Detection-main\Fake-News-Detection-main

pip install -r requirements.txt

python apps.py
